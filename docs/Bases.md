## Thème
Le package met à votre disposition plusieurs thèmes :
   
    - C Monochromatic
    - Pretty Saumon
    - Hunter X Hunter
    - Blue Night
    - Spider-man


## Variables Globales
Les variables disponibles dans le package sont les suivantes : 
  * Couleurs
  * Taille d’écran
  * Taille de texte 
  * Font de texte
  * Espacement
  * Bordure
  * Gras 
  * Opacité
  * Ombre
